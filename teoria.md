## Explica los siguentes conceptos con tus propias palabras. (no más de tres líneas por respuesta).

* HTML: es un lenguaje de codigo con el que se programan las paginas web necesario para que lo pueda leer el navegador 

* CSS: en el css se programan todas las caracteristicas visuales del codigo html.

* Git: git sirve trabajar en proyectos o archivos partiendo de un archivo maestro pudiendo ser modificado por por sus participantes en ramas las cuales pueden ser vueltas a unir al archivo maestro.  

* Arreglos: es un conjunto de numeros o strings las cuales tienen una posicion dentro del array.

* Objetos: es un conjunto de propiedades agrupadas en un objeto las cuales pueden contener valores,arrays u otros objetos

* Variables: son cajas donde se almacenan valores para poder ser usados cuando se invoquen.

* Métodos: son los que permiten manipular los objetos, los array, los strings de manera predefinida. 
ej : .length , .charAT(), .push 

* DOM: El DOM es la estructura de objetos que genera el navegador cuando se carga un documento y se puede alterar mediante Javascript para cambiar dinámicamente los contenidos y aspecto de la página

* Atributo de un tag HTML: son los que establecen sus propiedades, tamaño, fuente, color, posicion, etc

* Selector CSS: selecciona los elementos que coinciden con el selector nombrado ej: .class, #id, 

* Un función constructora: es la que establece las propiedades de un objeto de clase.
